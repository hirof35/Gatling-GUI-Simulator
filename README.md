# ガトリング砲スペック & 後座力シミュレーター (Gatling Gun Simulator)

男のロマンである「多砲身航空機関砲（ガトリング砲）」のスペックを比較し、射撃時の圧倒的な破壊力と凄まじい反動（後座力）を視覚的に体感できるPython/Tkinter製のGUIシミュレーターです。

## 💥 収録兵装
1. **GAU-8 Avenger (30mm)**
   - 搭載機：A-10 サンダーボルトII
   - 特徴：戦車を切り裂くための怪物砲。1秒間に約45kgの鉄の雨を降らせ、エンジンの推力を超える反動を叩き出します。
2. **M61 Vulcan (20mm)**
   - 搭載機：F-15, F-16, F-22 など
   - 特徴：米軍戦闘機に広く採用されている傑作20mm。毎分6,000発という圧倒的な連射速度を誇ります。
3. **GAU-22/A Equalizer (25mm)**
   - 搭載機：F-35 ライトニングII
   - 特徴：ステルス戦闘機用に軽量化（4砲身）された最新鋭。高威力ですが、機内装弾数180発というシビアな制限を持っています。

## 🚀 機能特徴
- **トリガー長押しシミュレート**
  「トリガーを引く」ボタンをマウスで押し続けている間、実弾さながらのスピードでマガジン残量（プログレスバー）が高速減少します。
- **リアルタイム物理計算**
  運動量保存の法則（$F = m \cdot v \cdot rps$）に基づき、**「1秒あたりの投射重量(kg/s)」**と**「平均反動(kgf)」**をリアルタイムに算出・表示します。

## 🛠 動作環境・実行方法
Python 3.x の標準ライブラリ（`tkinter`）のみで構築されているため、追加のパッケージインストール（`pip install` 等）は一切不要です。

### 実行コマンド
```bash
python GatlingGUISimulator.py
📝 物理計算の裏側（マニア向け）射撃時に発生する平均反動（後座力 $F$）は、以下の計算式で求められています。$$F = (\text{弾薬重量} \times \text{秒間発射数}) \times \text{砲口初速}$$これを地球の重力加速度（$9.80665 m/s^2$）で割ることで、直感的に分かりやすい「kgf（キログラム重）」単位に変換しています。A-10のGAU-8をぶっ放した際に出力される約4.6トン（4,600kgf）の反動を、ぜひ画面で確かめてみてください。📜 ライセンスMIT License / 改造・配布自由。トリガーハッピーなプログラミングライフを！
